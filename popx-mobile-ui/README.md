## PopX Mobile UI (React + Vite)

This project is a pixel-perfect implementation of the PopX mobile flow based on the provided Adobe XD design. It renders a centered mobile frame on desktop and includes navigation between Welcome, Login, Signup, and Profile screens using React Router.

### Tech Stack

- **Framework**: React (Vite)
- **Routing**: React Router
- **Styling**: Plain CSS with reusable components

### Getting Started

1. **Install dependencies**

```bash
npm install
```

2. **Run the development server**

```bash
npm run dev
```

3. Open the local URL printed in the terminal (usually `http://localhost:5173`) in your browser.

### Production Build

```bash
npm run build
npm run preview
```
